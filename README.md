# 🛒 E-Commerce User Behavior Analysis

This project analyzes user interactions with an e-commerce platform to understand behavior patterns, identify popular product categories, and improve conversion rates.

---

## 📁 Dataset

The dataset includes the following columns:

- **User ID** – Unique identifier for each user
- **Timestamp** – Date and time of the interaction
- **Action** – User action (`viewed_product`, `added_to_cart`, `purchased`)
- **Product Category** – Category of the product interacted with
- **Revenue** – Revenue generated (if applicable)

---

## ⚙️ Project Tasks

### 1. Data Cleaning & Preprocessing
- Handle missing values
- Convert timestamps to datetime format
- Extract time features (Date, Week, Hour)
- Derive category-level metrics

### 2. Analysis
- Identify the most popular product categories
- Calculate overall and category-wise conversion rates
- Visualize user activity trends over time

### 3. Insights & Recommendations
- Provide actionable insights based on user behavior
- Suggest ways to improve conversion rates
